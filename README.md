# Cybel Engine

The Cybel Engine is a simple, C++ game engine using SDL2, WebGL 2.0 (OpenGL ES 3.0), and legacy OpenGL 2.1+. It can build for the Desktop or build for the Web using Emscripten.

It was specifically written for my game [EkoScape](https://github.com/esotericpig/EkoScape), but can be used in general.

This repo is currently a work in progress for the next version of EkoScape, which can be viewed on its [dev branch](https://github.com/esotericpig/EkoScape/tree/dev).
