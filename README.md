# Cybel Engine

Cybel Engine is a simple C++20 game engine using SDL2, legacy OpenGL 2.1+, and WebGL 2.0 (OpenGL ES 3.0). It can build for the Desktop or the Web using Emscripten.

It was specifically written for my game [EkoScape](https://github.com/esotericpig/EkoScape), but not tied to it and can be used in general to make a game.

This repo is currently a work in progress for the next version of EkoScape, which can be viewed on its [dev branch](https://github.com/esotericpig/EkoScape/tree/dev).
